falling duck
===============

游戏介绍
---------
    您将创建一个duck玩家精灵，创建树桩障碍物。

    游戏规则：

     1. 游戏开始之后，duck往下掉，但是不能出屏幕
     2. 按下任意按键，duck往上，并且显示Jump的动画效果。
     3. 每间隔2s， 出现一个新的树桩，树桩匀速从右往左运动。
     4. 当duck碰到树桩，游戏结束
     5. duck每通过一个树桩，得分+1


游戏教程
---------

1. 选择【精灵】->【将精灵设为...】积木块，来创建一个duck精灵，形象可以自己设计，也可以用资料库中的造型。





注意 销毁的不是全局变量
