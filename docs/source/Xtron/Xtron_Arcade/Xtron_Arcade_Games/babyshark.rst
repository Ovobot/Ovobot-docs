Baby Shark
=============

游戏介绍
---------
海洋污染很严重，大海里有很多垃圾在危害着babyshark。我们的环保小卫士去大海里收集垃圾，保护babyshark.

游戏规则：1. 游戏开始,baby shark随意游动.控制玩家在水中游动，收集水中的垃圾。
         2. babyshark碰到海洋垃圾，游戏结束。
         3. 玩家收集到垃圾，得分+1.
         4. 玩家碰到babyshark, 生命值-1。
         5. 玩家抢到急救包，生命值+1。
         6. 生命值为0，游戏结束。
